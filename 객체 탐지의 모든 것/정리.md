- 이미지 내에서 객체의 바운딩 박스를 정의
객체 탐지 기술 2가지
1. 두단계 접근 방식
: R-CNN 으로 높은 정확도, 속도 느려서 실시간 처리에 한계
2. 단일 단계 접근 방식
: 이미지 내의 모든 객체를 한 번에 탐지. YOLO, SSD 존재. 빠른 속도로 객체를 탐지하여 실시간 응용에 적합 . BUT 정확도 낮음. YOLO는 이미지를 그리드로 나누고 각 그리드 셀에서 객체를 예측. SSD 는 여러 크기의 바운딩 박스를 사용하여 다양한 크기의 객체를 탐지할 수 있도록 설계.

-> 객체 탐지 모델 훈련 시 레이블이 붙은 데이터 셋 필요. EX) COCO, PASCAL VOC, ImageNet 존재.
최적화 알고리즘으론 Adam, sgd 존재

객체 탐지는 딥러닝의 발전과 밀접한 관련.
ResNet, VGG, Inception 같은 모델들이 객체 탐지 모델의 백본으로 사용되면서 성능이 비약적으로 향성. But 여러 문제 존재. 다양한 조명 조건, 객체의 크기와 모양, 배경의 복잡성 등은 객체 탐지의 정확도를 낮춤.

딥러닝 중 cnn 기술은 이미지 데이터를 다루는 데 최적화되어 있어 객체 탐지의 주된 방법으로 자리잡음. Cnn 은 이미지 특징을 자동으로 추출하고 이를 통해 객체를 인식하는 데 필요한 정보 제공.

과거에는, 객체 탐지 기술이 비교적 단순한 방법에 의존함. Ex) HOG, SVM . But 이는 다양한 조명 조건, 각도, 크기 변화에 약했음.

그러다, 2012년 AlexNet 의 성공으로 cnn 아키텍처들이 개발되며 r can, fast r cnn, faster r cnn등이 등장하며 객체 탐지 기술에도 큰 영향 미침.

이중 특히 , faster R-CNN는 주목할 만한 모델로, rpn 을 도입하여 객체 후보 영역을 신속하게 생성하고 이를 cnn 을 통해 처리하는 방식으로 작동함. 이는 객체 탐지의 시간 효율성이 크게 향상되었음.

이후 yolo와 ssd 같은 실시간 객체 탐지 모델이 등장하며 객체탐지 기술 발전에 이바지함. Yolo 는 이미지 전체를 한 번에 처리하녀 객체를 인식하는 방식으로 높은 속도와 정확성을 자랑함.

위에서 언급한 대규모 데이터셋의 발전으로 데이터의 양과 질이 향상되어 딥러닝 모델의 성능도 자얀스럽게 개선되었음.

- cnn의 원리와 구조
: 객체 탐지의 핵심은 이미지 내에서 객체의 존재를 인식하고 이들의 정확한 위치를 찾아내는 것이다. 그 중에서도 cnn 은 강력한 성능을 보여줌.

Cnn 구조는 크게 입력층, 합성곱층, 활성화층, 풀링층, 완전 연결층 등으로 나눌 수 있음.

1. 입력층
: 원본 이미지를 받아들이는 역할을 하며 일반적으로 rgb 이미지의 경우 3개의 색상 채널 가짐.
2. 합성곱층
: 필터(커널) 을 사용하여 이미지의 공간적 특징을 추출함. 일반적으로 relu 함수가 활성화 함수로 사용됨.
3. 풀링층
: 특징 맵의 차원을 줄여주는 역할, 이는 계산 효율성을 높이고 모델의 과적합 방지.
4. 완전 연결층
: cnn 의 마지막 단계로 추출된 특징을 바탕으로 최종적인 예측 수행.

Cnn 기반으로 한 객체의 탐지 모델은 크게 2가지 접근 방식으로 나눔.
1. 지역화 기반 방법
: 이미지에서 특정 객체를 탐지하고 그 위치를 박스로 표시함. Ex) r cnn, fast r cnn, faster r cnn.
R cnn 은 먼저 이미지에서 객체 후보 영역을 생성한 후, 각 후보 영역에 대해 cnn 을 적용하여 객체의 클래스와 위치를 예측함. Fast r cnn 은 이러한 과정을 개선하여, 후보 영역을 cnn 에 한 번만 통과시키고, 이후에 클래스와 위치를 예측함. Faster r cnn 은 후보 영역을 생성하는 단계까지 cnn 으로 통합하여 속도를 더욱 개선함.

2. 단일 단계 방법
: 이미지에서 직접적으로 객체를 탐지함. Ex) yolo, ssd. Ssd 는 yolo 의 아이디어를 발전시켜, 다양한 크기의 특징 맵을 활용하여 객체를 탐지함.

비전 트랜스포머와 같은 모델은 cnn 의 구조적 한계를 극복하고 더 광범위한 문맥 정보를 활용하여 객체 탐지의 정확도를 높이는 데 기여함.

- yolo 작동 방식
: Yolo의 기본 아이디어는 이미지를 s x s그리드로 나누고 각 그리드 셀에서 특정 수의 바운딩 박스를 예측. 각 바운딩 박스는 객체의 클래스 확률과 바운딩 박스의 위치를 나타내는 좌표 포함. Yolo 의 네트워크 구조는 cnn 을 기반으로 하며 주로 darknet이라고 불리는 프레임워크를 사용함. Darknet은 yolo의 효율성을 위해 경량화된 구조를 가지고 있으며 빠른 연산 속도를 제공함. Yolo 는 모델 훈련 과정에서 다양한 데이터셋을 활용하여 객체의 다양한 형태와 크기, 색상 등을 학습함. Yolo는 이전 방법들보다 더 높은 정확도로 객체 탐지 가능.

Yolo 는 또한 비최대 억제 기법을 사용하여 겹치는 바운딩박스를 제거하고 최종 결과를 정제함. Yolo 는 여러 버전이 있으며, yolo1에서 시작하여 yolo5까지 발전해왔습니다.

Yolo 의 장점 중 하나의 속도임. Yolo 는 실시간 객체 탐지에 최적화되어 있어 일반적 gpu 환경에서 높은 프레임 속도로 객체를 탐지. 이는 자율주행차, 드론, 실시간 감지 시스템 등에서 사용.
이는 작은 객체에 대한 탐지 성능이 떨어지는 경향이 한계임. 이를 해결하기 위해 yolo3에서는 fpn 구조를 도입하여 다양한 스케일의 객체를 보다 효과적으로 탐지할 수 있도록 개선.

- ssd 특징 및 장점
: ssd의 가장 큰 특징은 단일 샷 방식임. 한번의 신경망 추론으로 여러 객체를 동시에 탐지가능. 또한 다양한 크기의 객체를 탐지할 수 있는 능력가짐. Ssd 는 여러개의 다른 스케일과 비율로 구성된 앵커 박스를 사용함. VGGNet과 같은 기존의 강력한 신경망 아키텍처를 기반으로 하녀 특징 추출을 수행함. 실제 ssd 는 속도 측면에서 우월.또한 매우 유연한 구조를 가짐. 여러 다양한 네트워크 아키텍처와 결합가능. 사용자의 필요에 따라 성능 최적화 가능. 학습과 추론 과정이 비교적 간단.

- faster R-CNN 의 혁신적 접근
: 전통적인 객체 탐지 방법들은 일반적으로 2가지 단계 거침, 1번 이미지에서 후보 영역을 생성하고 2번 이런 후보 영역을 분류하여 최종적인 객체 위치를 결정하는 것. 이는 계산적으로 비효율적이며 느린 속도를 초래함. SO FASTER R CNN 은 이 두 단계를 통합하여 성능을 획기적으로 향상시킴.
가장 핵심은 RPN으로 이는 이미지의 중요한 부분을 빠르게 식별함. 이런 RPN 도입 덕분에 FASTER R CNN 은 이전의 방법들보다 훨씬 빠른 속도로 후보 영역을 생성함.
