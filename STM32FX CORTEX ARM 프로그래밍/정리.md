- stm32 의 nucleo보드는 mcu 의 최소 기능만으로 구성된 개발 보드로 연결되지 않은 나머지 핀들을 사용자가 자유롭게 사용할 수 있도록 확장핀 커넥터로 연결됨. Stm32 nucleo 보드의 장점은 복잡한 회로 수정 없이 다른 보드나 회로를 연결하여 빠른 기능 점검과 구현을 하는데 있음.

- FreeRTOS
멀티테스킹지원, 선점형, 비선점형 중 선택 가눙, task 수 제한 없고, 동적 task 생성 가능, priority로 task schedule 관리 가능, 여러 개의 task 중 어느 task를 언제 얼마동안 실행시킬지를 결정하는 것이 스케줄링인데 이 os는 선점형과 비선점형으로 제공함. 

특징
1. Portable 
다양한 시스템에 적용 가능
2. ROMable 
ROM에 전체 커널을 올릴 수 있다
3.Premmptive
선점형 커널을 가짐
4.multitasking
멀티테스킹 지원

실행시간 예측

FreeRTOS 로는 linux, 안드로이드 os 등이 있으나 이는 큰 용량의 메모리가 필요하며 작은 용량의 메모리 사용시 FreeRTOS가 

I2C, SPI 통신과 I2C LCD
- 사용자의 요구사항이 실시간으로 제품의 상태를 LCD혹은 LED 표시로 요구하는 경우가 많다. LED는 숫자를 표시하는 것은 쉬우나 문자까지 표시하기는 어렵다. 이경우 LCD를 사용하게 되는데 LCD 는 L2C로 제어할 경우 전원과 2개의 신호선만 있으면 돼서 편리하다.
- SPI통신은 전이중 통신 모드이며 마스터 슬레이브 모드로 동작하며 마스터가 데이터를 초기화함

USART통신
- 시리얼 통신은 펌웨어 시스템 개발에 있어 가장 많이 사용하는 통신 방식 중의 하나로 RS-485, RS-422은 1KM가 넘는 먼 거리 통신에 사용함. 가까운 시스템의 사이의 통신은 USART로 하는 것이 편리하다.

시리얼 통신(직렬) 은 대개 하나의 신호선을 이용하여 데이터를 주고받는 통신을 일컬어 지칭함. 데이터 전송은 일정한 시간 간격으로 전송하게됨
